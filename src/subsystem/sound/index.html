Sound Effects

Sound effects are created using SFXR.js and then turned into Wave DataURLs. This is done to reduce network load (1 load for the sound effects vs. 1 per sound file). The effects are loaded from audiosrcs.js into the SoundEffects object, which clones the effects so that multiple instances can be played at once.

The code to use the SoundEffects manager is very simple:

      var fx = SoundEffects,
          sound = {'explosion': 'data:audio/wav;base64,UklGRo6EAABXQVZFZm10I...'};

      fx.init({
        sounds: sounds, /* an object containing audio effect src strings, keyed by effect name */
        callback: function() { /* effects are ready to be played now... */}
      });

      fx.play('explosion');
